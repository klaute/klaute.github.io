40040 · Secure AI Engineering

Integrating AI systems securely into engineering processes.

From local LLMs and RAG to privacy, source code protection, security by design and architecture.

Security

Topics

  • Evaluate local LLMs and cloud services securely
  • Plan RAG systems on company knowledge
  • Protect privacy, confidentiality and source code
  • Anchor security by design in AI-assisted workflows
  • Prepare architecture and integration into engineering processes

Focus

Technical feasibility combined with security and privacy boundaries.

Outcome

A realistic target picture for local AI in your environment.

Description

This workshop clarifies how AI systems can be integrated securely into engineering processes. It covers local models, RAG, privacy, source code protection, security by design, architecture, operating models and integration effort.
